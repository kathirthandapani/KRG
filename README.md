## Features
- **Responsive Design**: Optimized for phones, PCs, and laptops.
- **Dynamic Pricing**: Syncs with live market rates.
- **Admin Dashboard**: Manage products, rates, and order history.
- **Secure Ordering**: OTP verification for material orders.
- **Dark Mode**: High-fidelity dark and light themes.

## Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS.
- **Backend**: Node.js, Express.
- **Database**: SQLite3.
- **Messaging**: Twilio API (for OTP verification).

## Getting Started

### Prerequisites
- Node.js (v14 or higher)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/kathirthandapani/KRG.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment
